## 1. 📖 系统简介

### 1.1 系统概述
**PDM系统**（Product Data Management）是一个专业的产品数据管理系统，旨在帮助企业高效管理产品全生命周期数据。

### 1.2 主要功能
- ✅ **产品管理** - 产品信息、技术参数、版本控制
- ✅ **文档管理** - 文档上传、版本控制、权限管理
- ✅ **BOM管理** - 物料清单、组件关系
- ✅ **变更管理** - 变更请求、审批流程
- ✅ **用户认证** - 登录/登出、权限控制
- ✅ **通知系统** - 系统消息通知

### 1.3 技术架构
- **后端框架**: Django 6.0
- **数据库**: MySQL 8.0
- **前端**: Bootstrap 5
- **部署环境**: Ubuntu 24.04 LTS

## 5. 📦 产品管理

### 5.1 产品列表
**访问路径**: 首页 → 产品管理 → 产品列表

![产品列表](https://via.placeholder.com/800x400/17a2b8/ffffff?text=产品列表页面)

### 5.2 创建新产品
**步骤**:
1. 点击**"创建新产品"**按钮
2. 填写产品信息：
   - 产品编码（如：P20260001）
   - 产品名称
   - 产品描述
   - 技术参数
   - 状态（草稿/发布/归档）
3. 点击**"保存"**按钮

### 5.3 产品详情
**功能**:
- 查看产品完整信息
- 查看关联文档
- 查看BOM清单
- 查看变更历史

### 5.4 产品编辑
**操作**:
1. 在产品列表中找到目标产品
2. 点击**"编辑"**按钮
3. 修改产品信息
4. 点击**"保存"**按钮

### 5.5 产品搜索
**搜索方式**:
- 按产品编码搜索
- 按产品名称搜索
- 按描述关键词搜索
- 按状态筛选

## 6. 📄 文档管理

### 6.1 文档列表
**访问路径**: 首页 → 文档管理 → 文档列表

![文档列表](https://via.placeholder.com/800x400/6f42c1/ffffff?text=文档列表页面)

### 6.2 上传新文档
**步骤**:
1. 点击**"上传新文档"**按钮
2. 选择关联产品
3. 选择文档类型
4. 填写文档信息：
   - 文档标题
   - 文档描述
   - 版本号
   - 状态
5. 选择文件上传
6. 点击**"保存"**按钮

### 6.3 文档版本控制
**功能**:
- 查看文档历史版本
- 下载指定版本
- 比较版本差异
- 恢复旧版本

### 6.4 文档权限
**权限级别**:
- **只读**: 只能查看和下载
- **编辑**: 可以上传新版本
- **管理**: 可以删除和设置权限

### 6.5 文档搜索
**搜索方式**:
- 按标题搜索
- 按描述搜索
- 按文档类型筛选
- 按关联产品筛选

## 8. 🔄 变更管理

### 8.1 变更请求列表
**访问路径**: 首页 → 变更管理 → 变更列表

![变更列表](https://via.placeholder.com/800x400/dc3545/ffffff?text=变更请求列表)

### 8.2 提交变更请求
**步骤**:
1. 点击**"提交变更请求"**按钮
2. 填写变更信息：
   - 变更标题
   - 变更描述
   - 变更类型（设计/工艺/文档）
   - 紧急程度（低/中/高）
   - 关联产品
3. 点击**"提交"**按钮

### 8.3 变更审批流程
**流程**:
1. **草稿**: 创建变更请求
2. **已提交**: 提交审批
3. **审批中**: 审批人审核
4. **已批准**: 审批通过
5. **已拒绝**: 审批不通过
6. **已实施**: 变更完成

### 8.4 变更跟踪
**跟踪内容**:
- 变更历史记录
- 审批意见
- 实施状态
- 影响分析

## 11. ❓ 常见问题

### 11.1 登录问题
**Q: 忘记密码怎么办？**
A: 请联系系统管理员重置密码

**Q: 登录后页面空白？**
A: 清除浏览器缓存（Ctrl+Shift+Delete）

**Q: 登录失败提示？**
A: 检查用户名和密码是否正确

### 11.2 操作问题
**Q: 上传文档失败？**
A: 检查文件大小（不超过10MB）和格式

**Q: 找不到产品？**
A: 使用搜索功能或检查筛选条件

**Q: 无法提交变更？**
A: 确保填写所有必填字段

### 11.3 系统问题
**Q: 页面加载缓慢？**
A: 检查网络连接，刷新页面

**Q: 功能无法使用？**
A: 联系技术支持

**Q: 数据丢失？**
A: 立即联系管理员
