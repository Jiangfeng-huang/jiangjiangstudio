## 📋 核心功能速查

### 1. **产品管理**
- **创建产品**: 首页 → 创建新产品
- **编辑产品**: 产品列表 → 点击产品 → 编辑
- **搜索产品**: 顶部搜索框

### 2. **文档管理**
- **上传文档**: 文档管理 → 上传新文档
- **版本控制**: 文档详情 → 历史版本
- **下载文档**: 文档列表 → 下载按钮

### 3. **BOM管理**
- **添加组件**: 产品详情 → BOM管理 → 添加
- **查看结构**: BOM列表 → 树状展示
- **导出BOM**: BOM管理 → 导出按钮

### 4. **变更管理**
- **提交变更**: 变更管理 → 提交变更请求
- **审批流程**: 变更详情 → 审批状态
- **跟踪变更**: 变更列表 → 状态筛选

## 🔑 常用操作

### **创建新产品**
1. 点击首页"创建新产品"
2. 填写产品编码、名称、描述
3. 设置技术参数
4. 点击"保存"

### **上传文档**
1. 点击"上传新文档"
2. 选择关联产品
3. 选择文档类型
4. 上传文件
5. 点击"保存"

### **提交变更**
1. 点击"提交变更请求"
2. 填写变更信息
3. 选择关联产品
4. 点击"提交"
